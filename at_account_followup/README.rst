=============================
Customer Follow Up Management
=============================

This module will add customer follow-up management in Odoo 17 Community Edition.

Installation
============

To install this module, download it and add it to your Odoo addons folder. Then log on to your Odoo server, enable debug mode, update the Apps list, and install the module.

Upgrade
=======

To upgrade, download the updated module, place it in your Odoo addons folder, restart the server, and upgrade the module from the Apps menu.

Configuration
=============

Configure follow-up levels as needed.
